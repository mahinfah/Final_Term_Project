Role 4 — Admin
Description
The Admin manages the entire hospital platform: staff accounts, doctor profiles, specializations, global policies, and all reporting. The Admin is the only user who can create doctor and staff accounts and approve new doctor registrations.
Features
•	Log in to an admin dashboard 
•   Showing today's total appointments, total registered patients, total active doctors    and    total    pending billing	
•	 Manage doctor accounts: add new doctors (create their user account and doctor profile), approve or reject    pending registrations
•   Edit doctor details, deactivate doctors/
•	Manage specializations: add, rename, and delete medical specializations/
•	Manage receptionist accounts: create, edit, deactivate receptionist accounts/
•	Manage patient accounts: search and view all patients, deactivate accounts flagged for misuse
•	Set global appointment policies: minimum cancellation notice period (hours), maximum advance booking window (days), default consultation fee
•	View all appointments across the platform with filters by doctor, date, status, and booking source (patient/receptionist)
•	View hospital-wide revenue report: total revenue per day/week/month broken down by doctor and specialization
•	View appointment volume report: busiest doctors, most in-demand specializations, peak appointment days and hours
•	View doctor performance report: consultation count, average patient rating, no-show rate per doctor
•	Handle patient complaints: view submitted complaints, mark as resolved with an admin response note
•	Set global appointment policies: minimum cancellation notice period (hours), maximum advance booking window (days), default consultation fee
•	View and manage the billing dashboard: total paid, total pending, and overdue bills
•	Export any report as a printable HTML table
